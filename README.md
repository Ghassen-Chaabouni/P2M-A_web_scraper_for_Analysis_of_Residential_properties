# P2M-A_web_scraper_for_Analysis_of_Residential_properties
### Scraping tayara.tn 
#### Install
1/
```
$ pip install pandas
$ pip install bs4
$ pip install requests
$ pip install selenium
```
2/
Download ChromeDriver from https://chromedriver.chromium.org/downloads
